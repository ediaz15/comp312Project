# COMP 312 – Assignment Documentation (OPEN ME)

This folder contains everything need and required for the COMP 312 project.

## Project Overview
FreshBooker is a simplified haircut booking web application inspired by platforms such as Booksy. The project was developed using Python (Flask), HTML, CSS, and JavaScript to demonstrate web application structure, templating, and basic user interaction.

## Contents of This Folder
- `MISSION_STATEMENT.md` – Project purpose and goals  
- `CODE_OF_CONDUCT.md` – Collaboration and behavior expectations  
- `CHANGES_OVERVIEW.md` – Summary of changes made to the project  
- `TESTING_PROCEDURE.md` – Manual testing steps and expected results  
- `TEAM_CONTRIBUTIONS.md` – Description of each team member’s role
